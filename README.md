## Run Locally

Clone the project

```bash
   git clone git@github.com:HlaingMinThan/Vue-Global-News-App.git news-app
```

Go to the project directory

```bash
  cd news-app
```

copy .env.example as .env

```bash
  cp .env.example .env
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm run serve
```

and then 

u can check on [(http://localhost:8081)](http://localhost:8081)
